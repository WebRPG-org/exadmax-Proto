# Proto

Proto é um projeto de RPG Maker inicialmente desenvolvido para desktop e, em seguida, adaptado para a web através de uma engine modificada e integração com jQuery. O objetivo é manter o charme e a mecânica de RPG clássico, oferecendo uma interface acessível via navegador.

## 📖 Descrição

- **Origem**: Criado originalmente no RPG Maker, aproveitando seu sistema de eventos, mapas e batalhas.
- **Evolução**: A engine foi modificada para extrair lógica e gráficos do RPG Maker, permitindo o uso de jQuery para manipulação de DOM e eventos.
- **Interface Web**: Toda a experiência do jogo roda em um navegador, sem necessidade de instalação de cliente específico.

## 🚀 Funcionalidades

- **Mapas interativos**: Navegação pelos cenários originalmente desenhados no RPG Maker.
- **Sistema de batalha**: Combates por turnos com UI customizada em jQuery.
- **Gestão de inventário**: Adição e remoção de itens com animações simples.
- **Diálogos e cutscenes**: Renderização de textos e escolhas de diálogo diretamente na página.

## 🛠 Tecnologias

- **RPG Maker** (origem de assets e lógica)  
- **JavaScript / jQuery** (interações, eventos e UI)  
- **HTML5 / CSS3** (estrutura e estilo)  
- **Engine personalizada**: Adaptada para rodar dentro do navegador, mantendo compatibilidade com os recursos originais.
